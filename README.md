# IOC Threat Pipeline

Threat intelligence engineering project using Python, Git, GitHub, and Elastic SIEM.

## Current Features

- Downloads OSINT IOC feeds
- Deduplicates indicators
- Combines feeds into a single output

## Planned Features

- IOC validation
- Structured JSON exports
- Confidence scoring
- Metadata tagging
- Feed attribution
- Hourly automation
- Elastic SIEM integration
- Indicator match rules

## Technologies

- Python
- VS Code
- Git
- GitHub
- Linux
- Elastic Security
